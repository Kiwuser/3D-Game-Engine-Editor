3d-game-engine-editor

A project to make an editor for thebennybox/bennyQBD's 3D Game Engine.

Download .zip Download .tar.gz View on GitHub

Welcome to the editor project for BennyQBD/thebennybox's 3D Game Engine
This project is not associated with BennyQBD/thebennybox
The project is lead by Kiwuser, and contributed to by dragonCASTjosh as well.

How to get the current version running
1. Download the master repo as either a zip, or clone it to your computer
2. Open the project in Eclipse (other will work too, but the repo only has a Eclipse project)
3. Open test.java in com.editor.gui.main, and click run
4. Profit!

Basic features of the editor

  • Double clicking on a GameObject will display an info pane (In the Hierarchy pane)
  • Hierarchy pane
  • Re-sizable windows
  • WYSIWYG engine pane

If you really are interested in the project, please keep an eye on it! it's in it's very early stages, but we hope to get many more features in the project!
